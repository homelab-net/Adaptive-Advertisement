# Repo-Native Document Exports

This folder contains repository-native text exports of the project source documents currently uploaded in the working context.

## Purpose

These exports make the current documentation baseline searchable and referenceable directly from the repository for implementation, review, and AI-assisted development.

## Export policy

- Source uploads remain the highest-fidelity originals.
- Repo-native exports are working copies for searchability and grounding.
- PDF exports preserve substantive text, but not original page layout, figures, or exact table geometry.
- DOCX exports preserve substantive text and simple tables, but not all Word-native formatting semantics.
- Where PR #1 introduced short source-note placeholders, these exports provide the fuller repo-native companion copies.

## Scope of this PR

- Full repo-native exports for the current authoritative source set.
- A delta report documenting what is exact, normalized, reduced, or still missing between the uploaded source set and the repository.
