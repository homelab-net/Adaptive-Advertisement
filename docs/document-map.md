# Source-to-Repository Document Map

This map records how the uploaded project documents were curated into this repository.

## Selection principles

1. Prefer the highest active revision.
2. Preserve authoritative documents as the implementation baseline.
3. Separate supporting guidance and living artifacts from controlling baselines.
4. Keep pending change-control artifacts visible, but do not treat them as authority until merged into a formal revision.
